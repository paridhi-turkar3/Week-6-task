# Blog Management REST API

## Project Description
This project is a RESTful API built using Spring Boot. 
It provides CRUD operations for blog posts with proper validation and error handling.

## Features
- RESTful endpoints for posts
- Spring Data JPA integration
- H2 in-memory database
- CRUD operations
- API testing using REST Client/Postman

## Technology Stack
- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- Hibernate
- H2 Database
- Maven

## API Endpoints

GET /posts  
POST /posts  
PUT /posts/{id}  
DELETE /posts/{id}

## How to Run

Clone the repository:

git clone https://github.com/paridhi-turkar3/Week-6-task.git

Run the application:

mvn spring-boot:run

Open browser:
http://localhost:8080/posts
